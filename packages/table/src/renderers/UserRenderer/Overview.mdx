# UserRenderer

The `UserRenderer` component is designed to display user information in a tabular context.
Depending on the user data, it conditionally renders an interactive `Popover` with additional user details.

## How to import

<Source code={`import { UserRender } from '@devo/genesys-ui-table';`} />

## Extended usage

When additional information is give trough `userMapping` context value, a `Popover` component involve the information.
`userMapping` map the object key with the `value` argument.
