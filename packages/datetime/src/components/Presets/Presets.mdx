<Heading.H1>Presets</Heading.H1>

Component used to display a list of preset date ranges that can be selected by the user.
The preset date ranges can be organized into categories, with each category having a label and a list of preset ranges.

<Heading.H2>How to import</Heading.H2>

<Source
  code={`
import { Presets } from '@devoinc/genesys-ui-datetime';
`}
/>

<Heading.H2>Basic usage</Heading.H2>

<Canvas of={Stories.Base} />

<Heading.H2>How to use</Heading.H2>

The Presets component accepts a prop called `presets` shaped as an array of objects that defines the list of preset ranges that are displayed.

The list can be either a one-level structure:

<Source
  code={`
[
    {
      value: {
        from: 'now() - 30s',
        to: 'now()',
      },
      label: 'Last 30 seconds',
    },
    {
      value: {
        from: 'now() - 1m',
        to: 'now()',
      },
      label: 'Last minute',
    },
],
      
`}
/>

or multi-level if categories are required:

<Source
  code={`
[
      {
        label: 'Relative to',
        options: [
          {
            value: {
              from: 'now() - 30s',
              to: 'now()',
            },
            label: 'Last 30 seconds',
          },
          {
            value: {
              from: 'now() - 1m',
              to: 'now()',
            },
            label: 'Last minute',
          },
        ],
      },
      {
        label: 'Snap to',
        options: [
          {
            value: {
              from: 'now() @ 1m',
              to: 'now()',
            },
            label: 'Minute',
          },
         
        ],
      },
    ]
`}
/>

The Presets component also accepts a prop called `value` which is an object representing the currently selected date range.
The value object has a `from` and a `to` property.
