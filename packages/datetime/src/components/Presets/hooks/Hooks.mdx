<Heading.H1>Presets Hooks</Heading.H1>

These hooks provide a pre-defined set of presets to be used in the Preset component.
Each hook returns a list of preset date ranges.

<Heading.H2>Default Presets Hook</Heading.H2>

Defines an array of default presets.
This hook returns a set of time ranges for two given categories: "Relative to" and "Snap to".

<Heading.H4>Usage</Heading.H4>

<Source
  code={`
import { useDefaultPresets } from '@devoinc/genesys-ui-datetime';
  
const presets = useDefaultPresets();

return <Presets presets={presets} />;

`}
/>
