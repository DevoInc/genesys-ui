# DateTimeFloatingPicker

The `DateTimeFloatingPicker` is an enhancement of the `DateTime` component,
designed to offer extended functionality.
It presents a date type input field that, upon interaction, displays the
`DateTime` component in a floating form (using a `Popover` component),
enabling users to conveniently select a date and time through various means.

Users can manipulate the value either directly from the `DateTime` component
or via `DateTimeInput` component (input text field).

Each time the input text field value is modified, it undergoes validation
against accepted date formats.
The first value in the `dateFormats` array serves as the default value to display.
If no format is specified, the default value will be `YYYY-mm-dd HH:mm:ss`.

When the `Enter` key is pressed the `onApply` method is called (if it exists).
When the `Escape` key is pressed the `onCancel` method is called (if it exists).

Any change made to the `DateTime` component triggers the `onChange` method.

Most of cases you only need to set the `value` property and perform some action
with the `onchange` method.
