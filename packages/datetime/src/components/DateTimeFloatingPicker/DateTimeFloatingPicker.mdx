# DateTimeFloatingPicker

The DateTimeFloatingPicker is an extension of the DateTime component.
It renders a date type input that shows the aforementioned DateTime component when clicked, allowing users to select
a date and time in several ways.

## How to import

<Source
  code={`import { DateTimeFloatingPicker } from '@devo/genesys-ui-datetime';`}
