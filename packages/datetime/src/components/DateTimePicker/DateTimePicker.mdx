# [DateTimePicker](#datetimepicker)

The DateTimePicker is an extension of the DateTime component.
It renders a date type input that shows the aforementioned DateTime component when clicked, allowing users to select a date and time in several ways.

## [How to import](#how-to-import)

<Source
  code={`
import { DateTimePicker } from '@devo/genesys-ui-datetime';
`}
