## I18n

- `i18n.nextMonth`: Text for tooltip of the button to go to the next month
- `i18n.prevMonth`: Text for tooltip of the button to go to the previous month
- `i18n.inputMonth`: Text for tooltip of the button to select the month

## Hooks

### `useMonthSelector`

This hook is for use with a single calendar view

### `useMonthSelectorRange`

This hook is for use with a dual calendar view
