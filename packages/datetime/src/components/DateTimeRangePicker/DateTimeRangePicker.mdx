# [DateTimeRangePicker](#datetimerangepicker)

The DateTimeRangePicker is an extension of the DateTimeRange component.
It renders a date type input which shows the aforementioned DateTimeRange component when clicked, allowing users to select a date and time range in several ways.

## [How to import](#how-to-import)

<Source
  code={`
import { DateTimeRangePicker } from '@devo/genesys-ui-datetime';
`}
