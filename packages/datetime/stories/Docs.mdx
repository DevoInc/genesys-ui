## Working with UTC

All the components work in UTC if you want to use TZ you should transform the
dates by yourself previously, that way allow you to use whatever library you
want, here is an example using `date-fns-tz`:

<Source code={`import { zonedTimeToUtc, utcToZonedTime } from 'date-fns-tz';
...
// Before use in the component
const dateInZone = utcToZonedTime(date, "Europe/Madrid");
// After change in component
const dateInUtc = zonedTimeToUtc(dateInZone, "Europe/Madrid");
...`} />
