# AppBar

## Overview

Component used for creating a navigation bar. It can be customized to fit various use cases.

## How to import

<Source
  code={`
import { AppBar } from '@devoinc/genesys-ui';
`}
/>

## Examples

### Base

<Canvas of={Stories.Base} />

### With heading and actions

You can specify a custom heading using the `heading` prop or add custom
actions at the right side of the `AppBar`. `actions` prop receive an array
of `Buttons`or `IconButton`.

<Canvas of={Stories.WithHeadingAndActions} />

### With options

`options` are the same of `actions` prop but adding a `Divider` at the beging.
`options` prop receive an array of `Buttons`or `IconButton`.

<Canvas of={Stories.WithOptions} />

### Custom content

You can create a custom `AppBar`content using the subcomponents:

- `AppBar.Container`
- `AppBar.Navigation`
- `AppBar.Actions`
- `AppBar.Options`
- `AppBar.Heading`
- `AppBar.Divider`

<Canvas of={Stories.Custom} />
