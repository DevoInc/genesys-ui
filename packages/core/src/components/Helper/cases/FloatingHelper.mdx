# FloatingHelper

## Overview

The FloatingHelper give contextual and floating information to the
users in order to fill in a field or a form, configure a functionality... etc.
and also provide feedback about the status of processes or configurations.

## How to import

<Source
  code={`
import { FloatingHelper } from '@devoinc/genesys-ui';
`}
