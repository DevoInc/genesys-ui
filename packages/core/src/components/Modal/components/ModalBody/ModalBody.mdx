# Modal.Body

Panel Body of the Modal component which renders the main content, and manage the scroll behavior.

It's based on `Panel.Body` component and its interface and usage is almost the same.

## How to import

<Source code={`import { Modal } from '@devoinc/genesys-ui';`} />

## Basic usage

Defined with dot notation as `Modal.Body`, it must be used only as a child of a `Modal` or `Modal` based component.
