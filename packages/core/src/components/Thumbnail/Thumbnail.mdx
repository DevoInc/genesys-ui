<Heading.H2>Cases</Heading.H2>

When using `size="fullWidth"` the component instance needs to be wrapped
inside an element with it CSS width property defined. Otherwise the Thumbnail won't be visible.

Without a wrapper

<Source
  code={`
<Thumbnail size={'fullWidth'} img={'https://picsum.photos/200/300'} />
`}
/>

With a wrapper

<Source
  code={`
<div
  style={{
    width: '90px',
    height: '90px',
  }}
>
  <Thumbnail size={'fullWidth'} img={'https://picsum.photos/200/300'} />
</div>
`}
/>
