<H1>ContextualScrollLoader</H1>

<H2>Overview</H2>

The ContextualScrollLoader is a specific type of Loader component designed to be used with the infinite scroll functionality, such as in a Table that loads new rows from the backend while scrolling.
It provides a temporary overlay that is positioned relative to a container and offers the same options for loading indicators as the Contextual Loader - including both spinner and progress bar types.
