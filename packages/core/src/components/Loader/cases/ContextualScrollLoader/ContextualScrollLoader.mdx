# ContextualScrollLoader

## Overview

The **ContextualScrollLoader** is a case component of the Loader which should be
used with the infinite scroll functionality. E.g. in a Table which loads new
rows from the backend in scrolling and needs some time to do it.

## How to import

<Source
  code={`
import { ContextualScrollLoader } from '@devoinc/genesys-ui';
`}
