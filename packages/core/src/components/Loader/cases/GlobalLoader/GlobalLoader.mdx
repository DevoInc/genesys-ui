# GlobalLoader

## Overview

The **GlobalLoader** is a case component of the Loader which should be used
for global loads, such as when refreshing the browser, when doing login / logout,
when changing sections on the platform, etc.

## How to import

<Source
  code={`
import { GlobalLoader } from '@devoinc/genesys-ui';
`}
