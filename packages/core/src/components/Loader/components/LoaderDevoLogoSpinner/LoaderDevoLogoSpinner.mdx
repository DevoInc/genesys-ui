# Loader.DevoLogoSpinner

Devo Logo spinner of the Loader component when we want a global loading transition.

## How to import

<Source code={`import { Loader } from '@devoinc/genesys-ui';`} />

## Basic usage

Defined with dot notation as `Loader.DevoLogoSpinner`, this component should be used only as a child of a
`Loader` or `Loader` based component.
