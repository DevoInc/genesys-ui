# Loader.GradientContainer

The gradient container overlay of the Loader component when we want a contextual loading transition, but related with
hidden contents usually linked with scroll .

## How to import

<Source code={`import { Loader } from '@devoinc/genesys-ui';`} />

## Basic usage

Defined with dot notation as `Loader.GradientContainer`, this component should be used only as a child of a
`Loader` or `Loader` based component.
