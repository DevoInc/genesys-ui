# Loader.ProgressBar

Circular progress bar of the Loader component when we want a contextual loading transition, and we know
the state in every moment.

## How to import

<Source code={`import { Loader } from '@devoinc/genesys-ui';`} />

## Basic usage

Defined with dot notation as `Loader.ProgressBar`, this component should be used only as a child of a
`Loader` or `Loader` based component.
