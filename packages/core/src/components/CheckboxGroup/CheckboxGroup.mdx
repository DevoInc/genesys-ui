# CheckboxGroup

Component used to render a group of checkboxes with customizable properties.
The component takes in a set of child Checkbox components,
and allows for additional customization like direction, legend, or status.

## How to import

<Source code={`import { CheckboxGroup } from '@devo/genesys-ui';`} />
