# FlexItem

## Overview

Use `FlexItem` to wrap each child of Flex container components as `Flex`, `HFlex`,
`VFlex` and `Wrap` when you need specific layout behavior for any of them.

## How to import

<Source
  code={`
import { FlexItem } from '@devoinc/genesys-ui';
`}
/>

## Basic usage

<Canvas of={Stories.Base} />

`FlexItem` allows us to get the spacing and flex styles (flex, align-self, order, width, margin... etc.)
from its own props.

<Canvas of={FlexStories.FlexAndFlexItems} />

More info about using `FlexItem` components as children of flex container
components in the [Flex Structure story](?path=/docs/public-atoms-flex-docs--page#structure).

Likewise, its `as` property allows the component to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

<Source
  code={`
import * as React from 'react';
import { Flex, FlexItem } from '@devoinc/genesys-ui';

const MyFlexCmp = () => (

<Flex as={'ul'}>
  <FlexItem as={'li'}>Child 1</FlexItem>
  <FlexItem as={'li'}>Child 2</FlexItem>
  <FlexItem as={'li'}>Child 3</FlexItem>
</Flex>
); `} />
