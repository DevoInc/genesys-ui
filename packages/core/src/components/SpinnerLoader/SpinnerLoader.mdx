# [SpinnerLoader](#spinnerloader)

## [Overview](#overview)

The **SpinnerLoader** component is a loading type element that shows to the user
that a time should be waited while an underlying action is being processed.

This element is always used within a component, such as: Buttons, IconButtons or
Overlays.
