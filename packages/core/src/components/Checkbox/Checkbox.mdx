# Checkbox

Form element that allows users to make a binary choice, i.e.
a choice between one of two possible mutually exclusive options.
It provides a range of customization
options to control the appearance and behavior of the checkbox,
as well as additional props for accessibility and guidance.

Checkbox is a wrapper over the
[CheckboxControl](?path=/docs/components-core-form-checkboxcontrol--docs)
component, and uses the [Field](?path=/docs/components-core-form-field--docs)
component to provide additional functionality such as labeling, helper error
messages, and validation.
