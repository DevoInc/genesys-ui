# AppLayout

Component used to create a layout for an application. It can be customized to
fit various use cases.

## How to import

<Source code={`import { AppLayout } from '@devoinc/genesys-ui';`} />

## Basic usage

The most simple use of the `AppLayout` is using its components:

- `AppLayout`: it's the outer container.
- `AppLayout.Bar`: it's the top bar area.
- `AppLayout.Lead`: it's area just below the top bar, usually used for secondary navigation.
- `AppLayout.Content`: it's the content area.
