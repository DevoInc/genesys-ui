# AppLayout

## Overview

Component used to create a layout for an application. It can be customized to
fit various use cases.

## How to import

<Source code={`import { AppLayout } from '@devoinc/genesys-ui';`} />

## Examples

### Base

The most simple use of the `AppLayout` is using its components:

- `AppLayout.Container`
- `AppLayout.Bar`
- `AppLayout.Content`

<Canvas of={Stories.Base} />

### Double navigation

With the `AppLayout.Lead` component we can create a sub-level of menu for our
applications

<Canvas of={Stories.DoubleNavigation} />

### Content padding

You can pass to the `AppLayout.Content` component the property `padding` for set
your own padding

<Canvas of={Stories.ContentPadding} />

### No header

If you don't need a navigation bar you could pass only `AppLayout.Content` and
get directly the content

<Canvas of={Stories.NoHeader} />
