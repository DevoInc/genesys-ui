# [InputControl](#inputcontrol)

## [Overview](#overview)

Text inputs enable users to enter free-form text data. The type of text field
used should reflect the length of the content you expect the user to enter.

## [How to import](#how-to-import)

<Source
  code={`
import { InputControl } from '@devoinc/genesys-ui';
`}
/>

## [Basic usage](#basic-usage)

<Canvas of={Stories.Base} />

You can use `status` props to inform some behavior.

<Source
  code={`
<InputControl
  aria-label="Status base"
  id="status-base"
  placeholder="Status base"
  status={'base'}
/>
`}
/>

You can specify the size of the input control with the `size` props.

<Source
  code={`
<InputControl
  id="size-sm"
  aria-label="Size sm"
  placeholder="Size sm"
  size={'sm'}
/>
`}
/>

You can specify the size of the input element with the `inputWidth` props.

<Source
  code={`
<InputControl
  id="input-width-xxs"
  aria-label="Input width xxs"
  inputWidth={'xxs'}
  placeholder="XXS"
/>
`}
/>

You can add an addon block to the beginning of the input control, to the end
or to the both sides..

<Source
  code={`
<InputControl
  addonToLeft="www."
  addonToRight=".eu"
  aria-label="Addon to both sides"
  id="addonToBothWithButton"
/>
`}
/>

Yo can specify different types for the HTML input element.

<Source
  code={`
<InputControl aria-label="Type number" id="typeNumber" type="number" />
`}
/>
