<Heading.H1>Subcomponents</Heading.H1>

Set of subcomponents that can be used to build a `Menu` component.

<Heading.H2>Item</Heading.H2>

The `Menu.Item` component is used to define each interactive and actionable menu element.

<Heading.H3>Basic usage</Heading.H3>

<Canvas of={ItemStories.Base} />

<Heading.H3>Append and prepend content</Heading.H3>

You can add at the beginning and at the end of the menu item custom content, in addition to
the available icon and label blocks.

<Canvas of={ItemStories.WithAppendAndPrependContent} />

<Heading.H3>Selectable</Heading.H3>

The `Menu.Item` can be used as a selectable option. The role `menuitemcheckbox` or `menuitemradio`
is added automatically based in the `selectionScheme` prop value. In addition, a checkbox or radio
is rendered inside the item, but it's not visible, because its purpose is manage the selection behavior.

<Canvas of={ItemStories.Selectable} />

<Heading.H3>Custom content</Heading.H3>

You can add all the content of the `Menu.Item` by its prop `children`. You have to keep in mind
that the content probably will weed some extra space, specially vertical one. This way the mouse
events and selectable behavior still work.

<Canvas of={ItemStories.CustomContent} />
