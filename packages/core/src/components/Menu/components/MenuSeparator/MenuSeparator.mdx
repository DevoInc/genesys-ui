# Menu.Separator

This component is used to define a menu separator between menu items.

## How to import

<Source code={`import { Menu } from '@devoinc/genesys-ui';`} />

## Basic usage

This component is defined with dot notation as `Menu.Separator` and it should be a child of a `Menu` component.

## Props

There are no props available for this component.
