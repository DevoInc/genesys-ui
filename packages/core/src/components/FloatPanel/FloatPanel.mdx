# FloatPanel

## Overview

Component designed display content in a panel layout that appears in front of all other screen content.
This panel is resizable, and it's very easy to make it closable too.

## How to import

<Source code={`import { FloatPanel } from '@devoinc/genesys-ui';`} />

## Basic usage

It's used as a wrapper of other contents as its children.

<Canvas of={Stories.Base} story={{ height: '60rem' }} />

## Closable

There is a pre-defined component (`FloatPanel.Close`) to render the typical close button
at the top-right side of the panel.
