# Textarea

Component used to create a customizable text area for user input.
It provides a range of customization options to control the appearance and
behavior of the text area, as well as additional props for accessibility and
guidance.

Textarea is a wrapper over the [TextareaControl](?path=/docs/components-form-textareacontrol--overview) component, and uses the [Field](?path=/docs/components-form-field--overview)
component to provide additional functionality such as labeling, helper error
messages, and validation.
