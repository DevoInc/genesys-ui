# [Toast](#toast)

## [Overview](#overview)

`toast` is a wrapper around react-toastify emitter with some custom props to
improve styles and configure the content
[https://fkhadra.github.io/react-toastify/introduction/](https://fkhadra.github.io/react-toastify/introduction/)

**IMPORTANT!**

Remember to render the `ToastContainer` once in your application tree. If you
can't figure out where to put it, rendering it in the application root would be
the best bet.
[https://fkhadra.github.io/react-toastify/installation#the-gist](https://fkhadra.github.io/react-toastify/installation#the-gist)
