# EmptyState

This component is used to display informative messages when there is no content to show—such as empty search
results, empty lists, or load errors. It provides a reusable and flexible way to communicate
"no data" states to the user.

The component accepts customizable props for setting the main message (title), a secondary description (description),
and the visual style (format). It supports three visual formats that define how the message is rendered—from simple text
to a more prominent layout with icons and enhanced styling.
