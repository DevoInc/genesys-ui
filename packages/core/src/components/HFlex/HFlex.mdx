<H1>HFlex</H1>

<H2>Overview</H2>

Use `HFlex` to create a stacked group of children components distributed
in a single row. Each children can be separated horizontally using the prop `spacing`.

<H2>How to import</H2>

<Source
  code={`
import { HFlex } from '@devoinc/genesys-ui';
`}

### [Using the as property](#using-the-as-property)

The `as` property allow these components to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

If you want to use the `Flex` component rendered with a different HTML tag, you can use the `as` property.
It allows these components to be rendered as any HTML tag, such as div, li, section, and so on.

<Source
  code={`
import * as React from 'react';
import { Flex, FlexItem } from '@devoinc/genesys-ui';

const MyFlexCmp = () => (

{' '}

  <FlexItem as={'li'}>Child 1</FlexItem>
  <FlexItem as={'li'}>Child 2</FlexItem>
  <FlexItem as={'li'}>Child 3</FlexItem>
</Flex>
); `} />
