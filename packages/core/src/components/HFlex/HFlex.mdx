# HFlex

Use this component to create a stacked group of child components distributed
in a single row. Each child can be separated horizontally using the prop
'spacing'.

This component is based on `Flex` component, so it has almost the same interface or API
then this last one.

## How to import

<Source code={`import { HFlex } from '@devoinc/genesys-ui';`} />

## Using HFlex.Item

We can use `HFlex.Item` component in a flex structure wrapping children to
assign flex, order, and align-self CSS styles to each child component. It's only
necessary when we need those specific styles for any child.

This component is exactly the same as `Flex.Item` component, so if you need more information you only have to access
to its [documentation page](?path=/docs/components-layout-flex-components-flexitem--overview).

Here is an example that demonstrates different CSS flex and align-self
settings for each child component of a Flex component:

## Using the as property

The `as` property allow these components to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

If you want to use the `HFlex` component rendered with a different HTML tag, you
can use the `as` property. It allows these components to be rendered as any HTML
tag, such as div, li, section, and so on.

<Source
  code={`
import * as React from 'react';
import { HFlex } from '@devoinc/genesys-ui';

const MyFlexCmp = () => (

  <HFlex as={'ul'}>
    <HFlex.Item as={'li'}>Child 1</Flex.Item>
    <HFlex.Item as={'li'}>Child 2</Flex.Item>
    <HFlex.Item as={'li'}>Child 3</Flex.Item>
  </HFlex>
);`} />
