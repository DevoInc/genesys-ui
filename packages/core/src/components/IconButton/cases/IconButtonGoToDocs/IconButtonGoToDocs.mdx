<H1>IconButtonGoToDocs</H1>

<H2>Overview</H2>

The IconButtonGoToDocs is a special type of IconButton used to link to any of our documentation pages.
It renders as an HTML `<a>` tag by default with its target attribute set to \_blank to open the link in a new tab.
