# [CheckboxControl](#checkboxcontrol)

## [Overview](#overview)

Form element that allows users to make a binary choice, i.e. a choice between one of two possible mutually exclusive options. 
By default, when a checkbox is selected, it displays a checkmark or a tick inside a square or a circle, indicating that the option has been chosen.
