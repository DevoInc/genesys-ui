# [Tooltip](#tooltip)

## [Overview](#overview)

The Tooltip component is used to provide additional information
or context to users when they interact with an element.
When the user hovers over, focuses on, or touches the element,
a small popup containing text appears.
This text can be used to provide helpful hints, explain functionality,
or give additional details about the element.
