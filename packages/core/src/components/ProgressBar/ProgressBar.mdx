## Basic usage

<Canvas of={Stories.Base} />

You can use the `ProgressBar` in mode circular.

<Source
  code={`
<ProgressBar 
  percent={75}
  type='circular'
  showInfo={true}
  progress='warning'
/>
`}
/>

Showing info

<Source
  code={`
<ProgressBar 
  percent={75}
  customInfo={{
    startInfo: 'This is the start info',
    endInfo: 'This is the end info',
  }}
  showInfo={true}
/>
`}
/>
