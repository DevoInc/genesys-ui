# [ProgressBar](#progressbar)

## [Overview](#overview)

The `ProgressBar` component communicate either an undefined waiting time or the duration of a process.
It inform users about the progress of ongoing tasks.

## [How to import](#how-to-import)

<Source
  code={`
import { ProgressBar } from '@devoinc/genesys-ui';
`}
/>

## [Basic usage](#basic-usage)

<Canvas of={Stories.Base} />

You can use the `ProgressBar` in mode circular.

<Source
  code={`
<ProgressBar 
  percent={75}
  type='circular'
  showInfo={true}
  progress='warning'
/>
`}
/>

Showing info

<Source
  code={`
<ProgressBar 
  percent={75}
  customInfo={{
    startInfo: 'This is the start info',
    endInfo: 'This is the end info',
  }}
  showInfo={true}
/>
`}
/>
