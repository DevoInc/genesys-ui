# Box

## How to import

<Source
  code={`
import { Box } from '@devoinc/genesys-ui';
`}
/>

Use `Box` component to wrap a functional component and apply
layout styles to it: margin, position, padding, overflow... etc.
Definitely it provides a shorthand way to pass layout styles via props to
another component.

This is an example of `Box`:

<Canvas of={Stories.Base} />

## as

Likewise, its `as` property allow this component to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

```
import * as React from 'react';
import { Box } from '@devoinc/genesys-ui';

const MyFlexCmp = () => <Box as={'span'}>Box content</Box>;
```
