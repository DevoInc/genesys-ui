# [Box](#box)

## [Overview](#overview)

Box is a layout component designed to simplify the application of layout styles such as margin, position, padding, and overflow to a functional component.
By wrapping another component with the Box component, you can easily pass layout styles via props, providing a shorthand approach to styling.

## [How to import](#how-to-import)

<Source
  code={`
import { Box } from '@devoinc/genesys-ui';
`}
/>

### [as](#as)

Likewise, its `as` property allow this component to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

<Source
  code={`

import { Box } from '@devoinc/genesys-ui';

const MyFlexCmp = () => <Box as={'span'}>Box content</Box>;
`}
