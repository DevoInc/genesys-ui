<H1>Row</H1>

<H2>Overview</H2>

Layout component used for creating responsive grid layouts. It has a similar purpose
to the Col component, but is used for defining rows in the grid layout.

By using the Row component along with other components such as Col and Container,
one can easily create flexible and responsive layouts that adapt to different screen sizes and device types.
