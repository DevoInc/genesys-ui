# InlineMessage

## Overview

Component designed to position a container in relation to a trigger. It uses the Popper component under the hood.

## How to import

<Source
  code={`
import { InlineMessage } from '@devoinc/genesys-ui';
`}
/>

The most frequent use of the InlineMessage component is to include an `InlineMessage.Panel` as a child.
