<Heading.H1>TagGroup</Heading.H1>

<Heading.H2>Overview</Heading.H2>

The TagGroup component is used to group and display multiple
Tag components together in a horizontal layout.
It is useful when displaying multiple tags related to a certain item or category.

The TagGroup component takes care of aligning the tags and
providing a consistent look and feel across all of them.
It also provides the ability to set a label for the group and adjust the size of the tags.
