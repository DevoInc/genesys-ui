# [VFlex](#vflex)

## [Overview](#overview)

Use `VFlex` to create a stacked group of children components distributed
in a single column. You can separate each child vertically with the prop `spacing`.

## [How to import](#how-to-import)

<Source
  code={`
import { VFlex } from '@devoinc/genesys-ui';
`}
/>

## [Cases](#cases)

### as

Likewise, its `as` property allow this component to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

<Source
  code={`
import * as React from 'react';
import { VFlex, FlexItem } from '@devoinc/genesys-ui';

const MyFlexCmp = () => (

<VFlex as={'ul'} spacing={'cmp-lg'}>
  <FlexItem as={'li'}>Child 1</FlexItem>
  <FlexItem as={'li'}>Child 2</FlexItem>
  <FlexItem as={'li'}>Child 3</FlexItem>
</VFlex>
); ); `} />
