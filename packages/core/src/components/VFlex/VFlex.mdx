# VFlex

## Overview

Use `VFlex` to create a stacked group of children components distributed
in a single column. Each child can be separated vertically using the prop `spacing`.

## How to import

<Source code={`import { VFlex } from '@devoinc/genesys-ui';`} />

## Using VFlex.Item

We can use `VFlex.Item` component in a flex structure wrapping children to assign flex,
order, and align-self CSS styles to each child component. It's only necessary when we need
those specific styles for any child.

Here is an example that demonstrates different CSS flex and align-self

## Using the as property

The `as` property allow these components to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

If you want to use the `Flex` component rendered with a different HTML tag, you can use the `as` property.
It allows these components to be rendered as any HTML tag, such as div, li, section, and so on.

<Source
  code={`
import * as React from 'react';
import { Flex } from '@devoinc/genesys-ui';

const MyFlexCmp = () => (

<Flex as={'ul'}>
  <Flex.Item as={'li'}>Child 1</Flex.Item>
  <Flex.Item as={'li'}>Child 2</Flex.Item>
  <Flex.Item as={'li'}>Child 3</Flex.Item>
</Flex>
); `} />
