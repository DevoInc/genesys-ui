# AppMenu.Separator

This component is used to define a separator between app menu items.

## How to import

<Source code={`import { AppMenu } from '@devoinc/genesys-ui';`} />

## Basic usage

This component is defined with dot notation as `AppMenu.Separator` and it should be a child of a `AppMenu` component.
