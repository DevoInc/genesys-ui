# EditableContent

Component created to wrap another typographic components and convert them into
editable contents.

You only have to wrap the content and save the changes through a state or
another mechanism based in events as `onFocus`, `onBlur`...etc.

Depending on the needs of your content, you can use the `as` prop to wrap with
a specific HTML tag as `<span>`, `<div>`, `<ins>`... etc.
