# Tabs.Divider

This component is used to separate items or the aside content from the tabs list block.

## How to import

<Source code={`import { Tabs } from '@devoinc/genesys-ui';`} />

## Basic usage

It's defined with dot notation as `Tabs.Divider`. This component should be used only as a
child of a `Tabs` or `Tabs` based component.
