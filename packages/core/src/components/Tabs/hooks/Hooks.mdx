# Tabs Hooks

Hook to build a TabsItem list.
Given an array of tabs config and common config props,
it returns an array of children to be rendered by the TabsContainer.

## UseTabsConfig

<Canvas of={Stories.Base} />

#### Parameters

<Source
  code={`
{ 
  active,    // Initial tab index activated
  href,      //
  iconId,    // 
  size,      // Default size
  state,     //
  tabs,      //
  target,    //
  wide,      //
}
`}
/>

#### Usage

<Source
code={`
import { Tabs, useTabsConfig } from '@devoinc/genesys-ui';

const tabs = [
{ label: 'Tiny' },
];
const tabsConfig = useTabsConfig({ tabs });

const children = tabsConfig.map((tab, idx) => (

<Tabs.Item {...tab} key={idx} />
)) as never;

return (

<Tabs>
  {children}
  <Tabs.Aside>Aside area</Tabs.Aside>
</Tabs>
);

`}
/>
