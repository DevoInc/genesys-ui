# Typography

## Overview

The `Typography` component can render any of the type block components: `Typography.Paragraph`, `Typography.Heading`,
`Typography.Mono`... etc. combining two props: 'format' and 'colorScheme'. The prop format is a quick way of defining the variant-size scheme
of the typographic element: body-sm, heading-h2, hero-sm... etc.

## Usage based in dot notation

With this component you can also define with dot notation several typographic block and inline components.

### Block components

You can read the documentation about this group of components [here](?path=/docs/components-text-typography-block--docs).

- Typography.BlockQuote
- Typography.Caption
- Typography.Code
- Typography.Lead
- Typography.Heading
- Typography.List
- Typography.ListItem
- Typography.Paragraph

<Source
  code={`
