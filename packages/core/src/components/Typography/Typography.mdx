# Typography

## Overview

The `Typography` component can render any of the type block components: `Typography.Paragraph`, `Typography.Heading`,
`Typography.Mono`... etc. combining two props: 'format' and 'colorScheme'. The prop format is a quick way of defining the variant-size scheme
of the typographic element: body-sm, heading-h2, hero-sm... etc.
