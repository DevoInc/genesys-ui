# Typography.BlockQuote

This component renders an HTML Block Quotation Element. It indicates
that the enclosed text is an extended quotation. A URL for the source of the
quotation may be given using the 'cite' attribute.

## How to import

<Source code={`import { Typography } from '@devoinc/genesys-ui';`} />

## Basic usage

It's defined with dot notation as `Typography.BlockQuote`, because this component belongs to `Typography` group.
