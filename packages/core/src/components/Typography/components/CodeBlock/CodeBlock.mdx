# Typography.CodeBlock

This component displays its contents styled in a fashion intended to
indicate that the text is a fragment of computer code.

<Source code={`import { Typography } from '@devoinc/genesys-ui';`} />

## Basic usage

It's defined with dot notation as `Typography.CodeBlock`, because this component belongs to `Typography` group.
