# Typography.List

List component can render an unordered list (by default,
represented as an HTML ul) or an ordered list (represented as an HTML
'ol').

## How to import

<Source code={`import { Typography } from '@devoinc/genesys-ui';`} />

## Basic usage

These lists typically display items as bullets or numbered items.

Each child component must be a `Typography.List.Item` which renders an HTML `<li>` element.

They are defined with dot notation as `Typography.List` and `Typography.List.Item`, because these components
belong to `Typography` group.
