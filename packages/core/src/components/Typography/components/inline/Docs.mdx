<Heading.H1>Inline components</Heading.H1>

Inline typographic components are typically used within other typographic block components.

They inherit the main text styles from those block components,
but they also have their own inline-specific styles, such as text-decoration, font-weight, and vertical-align.

<H2>How to import</H2>
<Source
  code={`
import { Typography } from '@devoinc/genesys-ui';
`}
/>

<Heading.H2>Abbr</Heading.H2>

`Typography.Abbr` renders an HTML Abbreviation element (`<abbr>`) which represents an
abbreviation or acronym. The optional 'tooltip' prop can provide an expansion
or description for the abbreviation. If tooltip is present, it must contain this
full description and nothing else.

<Heading.H2>Deleted</Heading.H2>

`Typography.Deleted` renders an HTML `<del>` element, which represents a range of text that has
been deleted from a document. This can be used when rendering "track changes"
or source code diff information, for example.

`Typography.Italic` renders an HTML `<em>` element which marks text that has stress emphasis.
The `Italic` component can be nested, with each level of nesting indicating a
greater degree of emphasis.
