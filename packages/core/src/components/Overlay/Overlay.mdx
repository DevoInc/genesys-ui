## [Overview](#overview)

The `Overlay` component provides an additional layer to the interface in order
to add another level of navigation into the app or blocking the screen to allow
an internal process to be ended.

The main properties of Overlay are:

- its area can occupy the whole page if it is **fixed** or contextual to a container
  (with relative position).
- it can be **opacity** or totally **opaque**.
- two main **scheme colors**: dark and light
