# Wrap

## Overview

Use `Wrap` to create a stacked group of children components distributed
in columns and rows. Each children can be separated horizontally using
the prop `hSpacing` and vertically using `vSpacing`.

## How to import

<Source
  code={`
import { Wrap } from '@devoinc/genesys-ui';
`}
/>

## Cases

### childrenByRow

You can use prop `childrenByRow` to define columns with the same width for each child:

<Source
  code={`
<Wrap childrenByRow={4}>
  ...
</Wrap>
`}
/>

### Using the as property

The `as` property allow these components to be rendered as any HTML tag:
'div', 'li', 'section'... etc.

If you want to use the `Flex` component rendered with a different HTML tag, you can use the `as` property.
It allows these components to be rendered as any HTML tag, such as div, li, section, and so on.

<Source
  code={`
import * as React from 'react';
import { Flex } from '@devoinc/genesys-ui';

const MyFlexCmp = () => (

{' '}

<Flex as={'ul'}>
  <Flex.Item as={'li'}>Child 1</Flex.Item>
  <Flex.Item as={'li'}>Child 2</Flex.Item>
  <Flex.Item as={'li'}>Child 3</Flex.Item>
</Flex>
); `} />
