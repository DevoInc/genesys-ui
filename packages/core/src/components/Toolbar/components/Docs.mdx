## Toolbar.Group

The `Toolbar.Group` component is used to split the toolbar component in different sections.

### How to use

<Source
  code={`
<Toolbar>
  <Toolbar.Group {...args}>
    <Toolbar.Item>One</Toolbar.Item>
  </Toolbar.Group>
  <Toolbar.Group>
    <Toolbar.Item>Two</Toolbar.Item>
  </Toolbar.Group>
</Toolbar>
`}
/>

## Toolbar.Item

The `Toolbar.Item` component is used to create items to be used inside a `Toolbar.Group`
or directly outside as child of the `Toolbar` component.

### How to use

<Source
  code={`
<Toolbar>
  <Toolbar.Group {...args}>
    <Toolbar.Item>One</Toolbar.Item>
    <Toolbar.Item>One</Toolbar.Item>
  </Toolbar.Group>
</Toolbar>
`}
/>

### Basic usage

<Canvas of={ItemStories.Item} />

<ArgTypes of={ItemStories} />

## Toolbar.Separator

The `Toolbar.Separator` component is used to create a separator mark between items.

### How to use

<Source
  code={`
<Toolbar>
  <Toolbar.Group>
    <Toolbar.Separator />
  </Toolbar.Group>
</Toolbar>
`}
