## [Toolbar.Group](#toolbar-group)

The `Toolbar.Group` component is used to split the toolbar component in different sections.

### [How to use](#how-to-use)

<Source
  code={`
<Toolbar>
  <Toolbar.Group {...args}>
    <Toolbar.Item>One</Toolbar.Item>
  </Toolbar.Group>
  <Toolbar.Group>
    <Toolbar.Item>Two</Toolbar.Item>
  </Toolbar.Group>
</Toolbar>
`}
