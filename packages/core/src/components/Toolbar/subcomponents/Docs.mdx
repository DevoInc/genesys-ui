<Heading.H2>Toolbar.Group</Heading.H2>

The `Toolbar.Group` component is used to split the toolbar component in different sections.

<Heading.H3>How to use</Heading.H3>

<Source
  code={`
<Toolbar>
  <Toolbar.Group {...args}>
    <Toolbar.Item>One</Toolbar.Item>
  </Toolbar.Group>
  <Toolbar.Group>
    <Toolbar.Item>Two</Toolbar.Item>
  </Toolbar.Group>
</Toolbar>
`}
/>

<Heading.H2>Toolbar.Item</Heading.H2>

The `Toolbar.Item` component is used to create items to be used inside a `Toolbar.Group`.

<Heading.H3>How to use</Heading.H3>

<Source
  code={`
<Toolbar>
  <Toolbar.Group {...args}>
    <Toolbar.Item>One</Toolbar.Item>
    <Toolbar.Item>One</Toolbar.Item>
  </Toolbar.Group>
</Toolbar>
`}
/>
