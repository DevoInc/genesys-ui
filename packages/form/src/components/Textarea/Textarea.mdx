<H1>Textarea</H1>

Component used to create a customizable text area for user input.
It provides a range of customization options to control the appearance and behavior of the text area,
as well as additional props for accessibility and guidance.

Textarea is a wrapper over the [TextareaControl](?path=/docs/components-core-form-textareacontrol--docs)
component, and uses the [Field](?path=/docs/components-core-form-field--docs)
component to provide additional functionality such as labelling, helper error messages, and validation.
