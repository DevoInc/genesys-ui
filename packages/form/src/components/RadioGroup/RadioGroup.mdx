# RadioGroup

Component used to render a group of radios with customizable properties.
The component takes in a set of child Radios components as its children prop,
and allows for additional customization through props like direction (which determines whether the checkboxes are arranged vertically or horizontally),
legend (which provides a label for the group of checkboxes), or status.
