<H1>UploadFiles</H1>

Component that handles upload of files. It is based on [FilePond](https://github.com/pqina/filepond) library
and some of its plugins. However, not all of its properties are can use
inside UploadFiles component.

<H2>How to import</H2>

<Source
  code={`
import { UploadFiles } from '@devoinc/genesys-ui-form';
`}
/>

<H2>Cases</H2>

<H3>Custom files extension</H3>

To specify the extension file allowed to upload you should use
`acceptedExtensions` property.

This property must be an array with the extension files allowed.

_Note_: note that the extensions _must_ have the dot at the beginning.

<H3>Custom files size</H3>

To limit the size of the files to be uploaded there exist following properties: `minFileSize`,
`maxFileSize` and `maxTotalFileSize`. They are based on
[File size validation](https://pqina.nl/filepond/docs/api/plugins/file-validate-size/)
plugin.
