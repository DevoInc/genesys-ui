# FieldsCombiner

Component used to create complex form layouts by combining different types of form fields in a single row or column.
It provides customization options through props to control the appearance and behavior of the combined fields.

## How to import

<Source
  code={`
import { FieldsCombiner } from '@devoinc/genesys-ui-form';
`}
/>
