# [Genesys UI Resources](#genesys-ui-resources)

Genesys UI relies on a number of open source projects maintained by [DevoInc](https://github.com/DevoInc/) to provide its functionality.

This page lists the main projects and relevant links.

## [Genesys Icons](#genesys-icons)

Genesys Icons is an icons library that provides a set of icons to be used in Genesys UI. The icons are available both as a React component and as a font.

The icons gallery is available in [Genesys Icons - Gallery](https://devoinc.github.io/genesys-icons/?path=/story/gallery--icons).

- [genesys-icons - Github](https://github.com/DevoInc/genesys-icons)
- [genesys-icons - NPM](https://www.npmjs.com/package/@devoinc/genesys-icons)

## [Genesys Brand Devo](#genesys-brand-devo)

Devo's main brand schemes of design tokens defined by Genesys Design System.
This package contains the brand schemes required to generate Devo's light and dark brands.

A preview of the generated tokens can be found in [Genesys Brand Devo - Tokens](https://devoinc.github.io/genesys-tokens/).

- [genesys-brand-devo - Github](https://github.com/DevoInc/genesys-icons)
- [genesys-brand-devo - NPM](https://www.npmjs.com/package/@devoinc/genesys-icons)

Genesys Brand Devo is generated using Genesys Tokens (see below).

## [Genesys Tokens](#genesys-tokens)

Tokens' definition and generation tools.

This project provides the tools to generate Devo's brand design tokens and its customizations.
It includes the agnostic source for the generation of the tokens, as described in Genesys Design System.
It uses `style-dictionary` under the hood.

- [genesys-tokens - Github](https://github.com/DevoInc/genesys-tokens)
- [genesys-tokens - NPM](https://www.npmjs.com/package/@devoinc/genesys-tokens)

## [Genesys Base Styles](#genesys-base-styles)

Genesys Base Styles is a set of CSS styles that provide a base for Genesys UI components. It includes the CSS reset, font family and font size, among others.

- [genesys-base-styles - Github](https://github.com/DevoInc/genesys-base-styles)
- [genesys-base-styles - NPM](https://www.npmjs.com/package/@devoinc/genesys-base-styles)
