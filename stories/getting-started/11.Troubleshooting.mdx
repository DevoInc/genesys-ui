# Troubleshooting

#### Style inconsistencies

If you happen to run into styles inconsistencies, make sure that you have gone through the [Installation](/docs/getting-started-installation--docs) steps,
more specifically the [CSS reset](/docs/getting-started-installation--docs#adding-required-base-styles) step.
